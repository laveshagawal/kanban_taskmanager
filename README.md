# 🚀 Kanban Task Manager

A modern, responsive **Kanban Board Web App** built using **HTML, CSS, and JavaScript**.  
It allows users to manage tasks efficiently with drag-and-drop functionality and persistent storage.

---

## ✨ Features

- 📝 Add, delete, and manage tasks
- 📦 Drag & Drop between columns (To Do, In Progress, Done)
- 💾 Local Storage support (data persists after refresh)
- 🔢 Live task count for each column
- 🎨 Smooth hover animations and modern UI
- 📱 Responsive design

---

## 🖥️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Storage:** LocalStorage (Browser)

---


## ⚙️ How It Works

- Tasks can be added using a modal popup  
- Each task is draggable and can be moved across columns  
- Task data is stored in the browser using LocalStorage  
- Counts update dynamically based on task movement
